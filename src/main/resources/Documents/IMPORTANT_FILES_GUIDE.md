## 💡 Cách Sử Dụng Tệp MD

### **Khi code JavaScript:**
```
1. Mở FRONTEND_GUIDE.md → tìm trang nào cần code
2. Mở JAVASCRIPT_GUIDE.md → copy template code
3. Mở HTML_COMPONENTS_TEMPLATE.md → copy HTML
4. Code theo template
```

### **Khi code API:**
```
1. Mở API_TEST_GUIDE.md → xem API endpoint
2. Code JavaScript để fetch API
3. Test bằng Postman (theo guide)
```

### **Khi có lỗi:**
```
1. Tìm file MD liên quan (vd: SIGNUP_FIXED_COMPLETE.md)
2. Đọc giải pháp
3. Áp dụng fix
```
